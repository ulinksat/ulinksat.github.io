Boot Mode

ULink 所提供的 SDR modem 是基於 ZedBoard 而來的，故其 boot mode 也延用 ZedBoard 五種，但在軟體開發和運用上僅使用 JTAG mode、QSPK mode 和 SD mode 三種

ULink 目前提供二種 SDR modem 基板，其選擇方式如下

MBO-147-A

基板編號見紅框處

本基板的 boot mode 選擇排針位置如下所示

請注意，因電路設計緣故，選擇時必須使用 ULinkSat 提供的 jumper (二 pin 間有串接電阻)

JTAG mode

QSPI mode

MBO-148-A

基板編號見紅框處

本基板目前只支援 JTAG mode 和QSPK mode 二種

本基板較爲緊湊，因此未如 MBO-147-A 般方便使用 jumper 選擇，爲方便開發階段切換 boot mode，可將其連接到排針，利用 jumper 切換
